## 🎓 Fluxo de Aprendizado Recomendado

### **Primeira Semana (Iniciante)**

**Dia 1-2:**
- [ ] Leia [Guia Rápido](GUIA_RAPIDO_SAFRA_CAFE.md) (5 min)
- [ ] Acesse o sistema
- [ ] Crie uma safra teste

**Dia 3-4:**
- [ ] Pratique criar orçamento
- [ ] Pratique registrar romaneio
- [ ] Leia partes do [Manual](MANUAL_SAFRA_CAFE_USUARIO.md) conforme faz

**Dia 5:**
- [ ] Processe um romaneio em beneficiamento
- [ ] Gere uma venda
- [ ] Se tiver dúvida, consulte [FAQ](FAQ_TROUBLESHOOTING.md)

---

### **Gerente/Supervisor**

**Semana 1:**
- [ ] Leia tudo ([Guia](GUIA_RAPIDO_SAFRA_CAFE.md) + [Manual](MANUAL_SAFRA_CAFE_USUARIO.md) + [FAQ](FAQ_TROUBLESHOOTING.md))
- [ ] Pratique você mesmo (crie uma safra completa)
- [ ] Prepare dados de exemplo para treinar

**Semana 2:**
- [ ] Siga [Checklist de Treinamento](CHECKLIST_TREINAMENTO.md)
- [ ] Treine seu primeiro usuário (45 min)
- [ ] Recolha feedback

**Semana 3+:**
- [ ] Treine o resto do time
- [ ] Acompanhe primeiros romaneios

## ❓ Perguntas Frequentes Sobre Documentação

**P: Por onde eu começo se sou usuário novo?**
R: Leia [Guia Rápido](GUIA_RAPIDO_SAFRA_CAFE.md) (5 min), depois entre no sistema e crie uma safra teste.

**P: Tenho uma dúvida específica, onde procuro?**
R: [FAQ](FAQ_TROUBLESHOOTING.md) tem respostas rápidas. Se não encontrar, [Manual Completo](MANUAL_SAFRA_CAFE_USUARIO.md) tem mais detalhes.

**P: Preciso treinar meu time, por onde começo?**
R: [Checklist de Treinamento](CHECKLIST_TREINAMENTO.md) é seu guia. Leia primeiro, prepare dados de exemplo, depois siga o roteiro.

**P: O que significam esses termos estranhos (romaneio, cereja, etc)?**
R: [FAQ](FAQ_TROUBLESHOOTING.md) tem "Glossário Mínimo" no final com explicações simples.

**P: Qual é a diferença entre [Manual](MANUAL_SAFRA_CAFE_USUARIO.md) e [Guia Rápido](GUIA_RAPIDO_SAFRA_CAFE.md)?**
R: **Guia Rápido** = resumido (5 min), mostra os passos principais. **Manual** = completo (20 min), com todos os detalhes.

**P: Posso imprimir os documentos?**
R: Sim! Recomendamos imprimir:
- [ ] Guia Rápido (para sala de treinamento)
- [ ] Checklist Treinamento (para facilitador)
- [ ] FAQ (para pendurar na parede)
