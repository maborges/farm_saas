# ⚡ Guia Rápido: Safra de Café do Início ao Fim

**Para:** Usuários não-técnicos | **Tempo:** 5 minutos de leitura

---

## 🎯 Os 5 Passos Principais

### 1️⃣ **CRIAR SAFRA** (1 minuto)
**Menu: Agricultura → ⑥ Resultados → Resumo de Safras**
Clique em "+ Nova Safra"

```
Preenche:
├─ Cultura: "Café"
├─ Talhão: Seleciona área da fazenda
├─ Ano Safra: 2025/2026
└─ Data Plantio: 15 de setembro

✅ Clica "Salvar"
```

---

### 2️⃣ **PLANEJAR GASTOS** (3 minutos)
**Menu: Agricultura → ① Planejamento → Orçamento de Safra**

```
Clica "+ Adicionar Item" para cada tipo de gasto:

Sementes/Mudas:     R$ 1.200
Adubo:              R$ 2.000
Defensivo:          R$   800
Combustível:        R$ 1.500
Mão-de-Obra:        R$ 4.000
Outros:             R$   500
                    ─────────
TOTAL:              R$ 10.000

🎯 Objetivo: Saber quanto vai gastar ANTES de começar
```

---

### 3️⃣ **REGISTRAR COLHEITA** (2 minutos)
**Menu: Agricultura → ③ Colheita → Romaneios (Colheita)**

```
Quando colher, clica "+ Novo Romaneio":

Data:               01 de junho
Talhão:             Talhão A1
Peso Bruto:         5.000 kg
Umidade:            50%
Sacas (60kg):       ~80 sacas

✅ Salva romaneio
```

---

### 4️⃣ **PROCESSAR CAFÉ** (5 minutos)
**Menu: Agricultura → ③ Colheita → Romaneios (Colheita)**
Na tabela, clique botão 🏭 "Processar" → Lote criado em **Agricultura → ④ Pós-Colheita → Beneficiamento**

```├─ Registra secagem (data início/fim)
├─ Registra peso final (kg beneficiado)
├─ Preenche perdas:
│  ├─ Perda Secagem: 500 kg
│  ├─ Perda Quebra: 150 kg
│  └─ Perda Defeito: 100 kg
└─ Muda Status → ARMAZENADO

🎯 Objetivo: Transformar café cereja em café pronto para vender
```

---

### 5️⃣ **GERAR VENDA** (1 minuto)
**Menu: Agricultura → ④ Pós-Colheita → Beneficiamento**
Clique em botão 🛒 "Gerar Venda" nos lotes com Status = ARMAZENADO

```
Quando café está pronto (Status = ARMAZENADO):

Clica botão 🛒 "Gerar Venda"

✅ Venda criada (Status: RASCUNHO)
   Pode preencher dados do comprador depois
```

---

## 📊 O Que Você Vê em Cada Etapa

| Etapa | Página | O Que Aparece |
|-------|--------|---------------|
| **1. Safra** | Safras | Lista de safras criadas |
| **2. Orçamento** | Orçamento | Quanto planejou gastar |
| **3. Colheita** | Romaneios | Cada dia que colheu |
| **4. Processamento** | Beneficiamento | Café em processamento |
| **5. Venda** | Comercialização | Vendas prontas |

---

## 🆘 Precisa de Ajuda?

**"Não consigo achar o botão X"**
→ Botões estão sempre no topo direito da seção ou em cada linha da tabela

**"Dei um valor errado"**
→ Clica no ✏️ (lápis) em qualquer linha para editar

**"Perdi a safra"**
→ Menu: Agricultura → ⑥ Resultados → Resumo de Safras → Procura pelo nome/ano

**"Como vejo quanto custou cada coisa?"**
→ Safra → "Financeiro" → Mostra orçado vs realizado

---

## 💡 Dicas Importantes

✅ **Sempre** preencha o orçamento antes de começar
✅ **Registre colheita** todos os dias (não deixa acumular)
✅ **Processe café rápido** após colher (no máximo 2 dias)
✅ **Guarde números** de romaneios (ex: ROM-001, ROM-002)
✅ **Verifique perdas** durante processamento (devem ser ~40% do peso)

❌ **Não** deixe lotes em RECEBIMENTO por muito tempo
❌ **Não** invente valores de umidade (use medidor)
❌ **Não** misture métodos (NATURAL + LAVADO) no mesmo lote
